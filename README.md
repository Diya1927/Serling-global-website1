## 📊 Analytics (Optional)

To track visitors, add Google Analytics:

1. **Create Google Analytics account** → [analytics.google.com](https://analytics.google.com)
2. **Get your tracking ID** (format: G-XXXXXXXXXX)
3. **Add to HTML** (before closing `</head>` tag):

```html
<!-- Google Analytics -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'G-XXXXXXXXXX');
</script>
```

## 🛠️ Troubleshooting

### **Site won't load**
- ✅ Check internet connection
- ✅ Clear browser cache (Ctrl+Shift+Delete)
- ✅ Try incognito window
- ✅ Check Netlify deployment status

### **Images not showing**
- ✅ Check internet (Unsplash images need connection)
- ✅ Check browser console (F12 → Console tab)
- ✅ `s-hassan.jpg` must be in same folder as HTML

### **Mobile menu not working**
- ✅ Clear cache and reload
- ✅ Try different browser
- ✅ Check JavaScript is enabled
- ✅ Check console for errors (F12)

### **Calendly button doesn't work**
- ✅ Check link is correct: `https://calendly.com/sterlingglobaluk/30min`
- ✅ Try opening in new tab
- ✅ Check your Calendly account is active
